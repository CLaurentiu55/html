.dockerignore file

Create a .dockerignore file in the same directory as your Dockerfile with following contetnt:

node_modules
npm-debug.log

This will prevent your local modules and debug logs from being copied onto your Docker image and possibly overwriting modules installed within your image.

Building your image

Go to directory that has your Dockerfile and run the following command to buil the Docker image. The -t flag lets you tag your image so it's easier to find later using the docker images command:

docker build -t <your username>/node-web-app .
