Creating a Dockerfile

Creating empty file called Dockerfile:

touch Dockerfile

Open the Dockerfile in your favorite text editor

The first thing we need to do is define from what image we want ro build from. Here we will use the latest LTS(long term support) version 12 of node avaible from the Docker hub:

FROM node:12

Next we create a directory to hold the application code inside image, this will be the working directory for your application:

# Create app directory

WORKDIR /usr/src/app
